# Micropython for Windows
I cloned https://github.com/micropython/micropython for this build!
# How to install
Install the exe and drag to C:\micropython
# How to use
Either open C:\micropython (or where you dragged the exe) with cmd.exe and run micropython.exe
\nOR
\nAdd to PATH using sysdm.cpl
